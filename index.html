Detroit: Become Human

About the Game

Detroit: Become Human is an interactive drama and action-adventure game developed by Quantic Dream. The game is set in a futuristic Detroit in the year 2038, where advanced androids are integrated into society, performing tasks like household chores, factory labor, and assisting the elderly. The story revolves around three main android protagonists—Kara, Connor, and Markus—who each experience a different path toward self-awareness and autonomy.
